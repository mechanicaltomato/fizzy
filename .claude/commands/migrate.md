Manage database migrations for Fizzy.

Common migration tasks:
- `bin/rails db:migrate` - Run pending migrations
- `bin/rails db:rollback` - Undo last migration
- `bin/rails db:migrate:status` - Check migration status
- `bin/rails db:reset` - Drop, create, load schema
- `bin/rails generate migration AddFieldToModel field:type` - Generate new migration
